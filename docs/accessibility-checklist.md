# Accessibility Verification Checklist
- Contrast: all text, borders, controls meet WCAG AA.
- Keyboard: full navigation, visible focus ring, no traps.
- Screen reader: labels/hints on all controls and dynamic status.
- Text scaling: layout remains usable at larger sizes.
- Motion: avoid unnecessary animation; respect reduce motion.
- Copyability: health report/log fields selectable and copyable.
