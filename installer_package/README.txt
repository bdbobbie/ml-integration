This package contains ML Integration v1.0.0 with automatic installation.

INSTALLATION INSTRUCTIONS:
1. Double-click this file to run the auto-installer
2. The installer will automatically:
   - Extract ML Integration.app
   - Install to /Applications folder
   - Set proper permissions
   - Handle macOS security settings
   - Clean up temporary files

3. Launch ML Integration from /Applications folder

SYSTEM REQUIREMENTS:
- macOS 12.0 or later
- Apple Silicon or Intel Mac
- 8GB RAM minimum (16GB recommended)
- Virtualization support enabled

SECURITY:
This app is signed with Apple Development certificate.
If Gatekeeper shows a warning, right-click the app and select "Open".
