# ML Integration

A comprehensive macOS application for Linux virtualization with seamless integration and management capabilities.

© 2026 TBDO Inc. All rights reserved.

## Features

- **VM Lifecycle Management**: Complete provisioning, installation, and lifecycle control
- **Integration Services**: Shared resources, launchers, and rootless Linux apps
- **Health & Auto-Heal**: Automated diagnostics and recovery mechanisms
- **Security & Credentials**: Keychain management and signature verification
- **Observability**: Comprehensive logging and performance monitoring
- **Cleanup Operations**: Complete uninstall with verification and reporting

## System Requirements

- macOS 12.0 or later
- Apple Silicon or Intel Mac
- 8GB RAM minimum (16GB recommended)
- Virtualization support enabled
